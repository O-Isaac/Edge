### FAQ

<div className="[&>ul]:list-none mt-4">
**Q**: How to make the Stamina/Health bars colored?
- **R**: _Rename **hudmenu colored.swf** to **hudmenu.swf** (either rename the original file or delete it beforehand)_ 

**Q**: How to set bars to top left position?
- **R**: _Rename **skyhud - top playerbars.txt** to **skyhud.txt** (either rename the original file or delete it beforehand)_

**Q**: How to get the CSM Tweenmenu?
- **R**: _Rename **Tweenmenu CSM.swf** to **tweenmenu.swf**_

**Q**: How to get the left-sided Sleep/Wait menu?
- **R**: _Rename sleepwaitmenu - **left sided.swf** to **sleepwaitmenu.swf**_

**Q**: How to get the 21:9 Favorites menu?
- **R**: _Rename **favoritesmenu 21-9.swf** to **favoritesmenu.swf** (either rename the original file or delete it beforehand)_

**Q**: MoreHUD preset is not working
- **R**: *You need reload **moreHUD***:
    <div className="bg-neutral-900 text-white w-fit p-5 rounded-2xl mt-2">
        <ul className="lg:[&>li]:text-base [&>li]:text-sm">
            <li><span className="text-lg text-amber-400">1.</span> Uninstall moreHUD via MCM</li>
            <li><span className="text-lg text-amber-400">2.</span> Load save without moreHUD esp active</li>
            <li><span className="text-lg text-amber-400">3.</span> Re-enabling moreHUD and load EdgeUI preset via MCM.</li>
        </ul>
    </div>
</div>
