## Frequently Asked Questions

<Card title="How to make the Stamina/Health bars colored?">
  Rename hudmenu colored.swf to hudmenu.swf (either rename the original file or delete it beforehand)
</Card>

<Card title="How to set bars to top left position?">
  Rename skyhud - top playerbars.txt to skyhud.txt (either rename the original file or delete it beforehand)
</Card>

<Card title="How to get the CSM Tweenmenu?">
  Rename Tweenmenu CSM.swf to tweenmenu.swf
</Card>

<Card title="How to get the left-sided Sleep/Wait menu?">
  Rename sleepwaitmenu - left sided.swf to sleepwaitmenu.swf
</Card>

<Card title="How to get the 21:9 Favorites menu?">
  Rename favoritesmenu 21-9.swf to favoritesmenu.swf (either rename the original file or delete it beforehand)
</Card>

<Card title="MoreHUD preset is not working">
  You need reload moreHUD:
    1. Uninstall moreHUD via MCM
    2. Load save without moreHUD esp active
    3. Re-enabling moreHUD and load EdgeUI preset via MCM.
</Card>
